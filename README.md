# Доступные скрипты

- Запуск в режиме разработки:

  ```sh
  npm run dev
  ```

- Сборка проекта:

  ```sh
  npm run build
  ```

- Запуск в режиме предварительного просмотра производственной сборки:

  ```sh
  npm run preview
  ```

- Запуск ESlint:
  ```sh
  npm run lint
  ```
